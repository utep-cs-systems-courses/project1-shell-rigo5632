# Lab 1:Shell



The purpose of this lab is to create a simple shell that has the following
features:

* Get user Input and display a prompt

* Execute commands

* Input/Output Redirection

* Basic Pipes

* Exit

* cd

* Background Process





## How to run:

To run this lab run the following command:

```bash

python3 shell.py

```

or

```bash

./shell.py

```
